Crumb is a native iOS application designed to streamline the home baking experience. It solves common friction points like inventory management ("Raid My Pantry"), hands-free instruction navigation ("Kitchen Mode"), and recipe organization.

## ğŸ“± Features
* **Kitchen Mode:** Hands-free voice navigation using Apple's Speech Framework so bakers don't touch screens with messy hands.
* **Raid My Pantry:** Integrated Spoonacular API (`/findByIngredients`) to generate recipes based on user inventory.
* **Smart Parsing:** Extracts structured recipe data from any web URL.
* **Social Feed:** Instagram-style community feed with asynchronous image loading.
* **Journal:** Supabase-backed persistence for logging bake ratings and notes.
